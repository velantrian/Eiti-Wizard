🔍 О системе ▼

Нажмите, чтобы увидеть архитектуру приложения. ИИ также получает эту информацию в системном контексте.

VELAN EITI v1.7.7 — PWA-приложение для работы с AI

Frontend: Чистый HTML/CSS/JS без фреймворков. Один файл index.html, встроенный CSS и JS.

Хранилище: IndexedDB — базы: chats, messages, memory, files, config. Данные живут в браузере.

AI провайдеры: Ollama (офлайн, localhost:11434), Claude (Anthropic), OpenAI, DeepSeek, Groq, OpenRouter.

Режимы: Чат (история сообщений по сессиям), Агент (пошаговые задачи), Память (key-value хранилище подсказок), История, Файлы, Настройки.

PWA: Service Worker (sw.js), манифест (manifest.json), устанавливается на экран. Офлайн работает через Ollama.

Темы: 13 цветовых тем (тёмные + светлые), переключаются через data-theme на body.

Файлы: Загрузка документов (PDF, TXT, DOCX, изображений), векторный поиск через чанки для RAG-контекста.

💡 ИИ видит эту информацию через системный промпт и может давать советы по улучшению приложения.
