⚡ Eiti Wizard

🚀 Progressive Web App для работы с AI чатами, агентами и файлами в офлайн-режиме через Ollama и внешние модели


---

🧠 О проекте

Eiti Wizard — это локальный AI workspace, который объединяет:

💬 умный чат

🤖 автономных AI-агентов

📎 работу с файлами и большими документами

🧠 локальную память и историю

⚡ поддержку локальных и облачных моделей


Это не просто чат.
Это попытка собрать AI-рабочую среду внутри браузера.


---

✨ Возможности

💬 Умный чат

Множественные чаты с историей

Поддержка Ollama (офлайн AI)

Подключение внешних API (OpenAI, DeepSeek, Claude и др.)

Озвучивание сообщений (Web TTS)

Копирование ответов в один клик

Быстрая работа без фреймворков



---

🤖 AI Агенты

Автономные агенты для выполнения задач

Контекст из файлов и чатов

История работы агентов

Возможность расширения под инструменты и действия



---

📎 Умная работа с файлами

Загрузка и хранение файлов

Разбиение на чанки (chunk-based processing)

Перекрытие чанков для сохранения контекста

Поиск релевантных частей документа

Поддержка разных форматов


👉 Это основа будущего “RAG внутри браузера”


---

🧠 Память системы

IndexedDB локальное хранение

История сообщений и чатов

Настройки и конфигурации

Подготовка к graph memory (в разработке)



---

🎨 Интерфейс

Темная / светлая тема

Glassmorphism (стеклянный эффект)

Адаптивный дизайн (mobile + desktop)

Полноэкранный режим (двойной тап)

---

📱 PWA режим

Установка как приложение
